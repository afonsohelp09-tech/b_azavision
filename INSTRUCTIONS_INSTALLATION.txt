ÉTAPES RAPIDES
  1. Créer ou ouvrir le projet Apps Script lié à votre tableur ERP (Extensions >
     Apps Script), coller api_apps_script.gs, enregistrer.
  2. Ouvrir le Google Sheets lié à ce script : un menu « ERP Vente en ligne »
     apparaît dans la barre (après rechargement de la page). Utilisez-le pour :
       • Initialiser les feuilles (première configuration) — crée onglets,
         compte admin admin@loja.com / admin123 si USERS vide, CONFIG / EMPRESA de base ;
       • Enregistrer la liaison avec ce classeur ;
       • Aide : rappel pour déployer l’API Web (/exec).
     Si le menu n’apparaît pas : Apps Script → sélectionner la fonction onOpenManualRun
     → Exécuter → autoriser, puis actualiser Sheets.
  3. Déployer → Application Web → Exécuter en tant que moi → Accès : Toute personne
     (Anyone) ; copier l’URL se terminant par /exec.
  4. Dans admin_erp.html et boutique_client.html : renseigner la MÊME URL dans
     window.ERP_API_URL_DEFAULT = '…' et/ou sur la ligne marquée COLAR (les deux
     peuvent contenir le même texte ; localStorage garde la priorité après un test
     de connexion depuis l’écran admin).
  5. Héberger les HTML en HTTPS (Google Sites « Incorporer », hébergeur, Drive partagé
     avec bons CORS si besoin) ; éviter d tester en file:// pour les appels fetch.
